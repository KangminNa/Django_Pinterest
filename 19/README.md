# 19. Pycharm 디버깅 설정

## 공부
- 파이참에서 서버를 디버깅 하는 방법에 대한 이야기를 시작!

- 코드가 많기 때문에 중단점을 두고 Debugging을 하는 설정을 해보려고 함.
- run tab을 누르면 edit configurations 클릭
